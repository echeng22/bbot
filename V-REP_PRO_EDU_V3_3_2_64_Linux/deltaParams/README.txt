Variable Definition for Delta Robot
---------------------------------------------------------------------
r1: Radius of Base
r2: Radius of End Effector
l1: Length of Lower Link
l2: Length of Upper Link

----------------------------------------------------------------------
Model can be saved by typing in desired file name and using the save
model button. The model will be saved in the deltaModels folder where
the V-REP installation folder is found. If the folder does not exist when
trying to save this model, a folder will be automatically created. This folder
will be used in other scripts for loading delta robot models for this project.
